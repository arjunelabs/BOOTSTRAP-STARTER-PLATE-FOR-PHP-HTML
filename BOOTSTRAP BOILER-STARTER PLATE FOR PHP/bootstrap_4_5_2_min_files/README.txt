 * This is a boiler Plate for making any BOOTSTRAP Project.
 
 * This folder contains all the required files that needs to build the bootstrap project.
 
 * Kindly copy this folder and paste it into your project directory.
 
 * You can also use the HTML file which contains all the bootstrap supported files linked to it.
 
 * All the files are minified version so execution of the files will be very fast.
 
 * This makes this folder ready to start your project without worrying to link
 or attach the bootstrap files ot it.
 
 * Bootstrap Folder contains 5 files :
 
 1. Fontawesome.min.css
 2. Bootstrap.min.css
 3. Bootstrap.min.css.map
 4. jQuery.min.js
 5. Popper.min.js
 6. Bootstrap.min.js
 7. Bootstrap.min.js.map
